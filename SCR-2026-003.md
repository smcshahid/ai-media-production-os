# Scope Change Request — Audio Narration Pilot

**SCR ID:** SCR-2026-003

**Baseline release:** `v0.14.0-phase4-multiscene`

## Summary

Authorize sovereign **audio narration** for scene videos: local TTS (espeak-ng), FFmpeg mux into MP4, **NARRATION** asset storage, **export manifest v3**, with silent-video fallback and v1/v2 backward compatibility.

---

## Scope

| In scope | Out of scope |
|----------|--------------|
| TTS within VIDEO activity | Character bible |
| NARRATION asset stage | Episode model |
| Manifest v3 | Publishing APIs |
| Review narration badge | OAuth/RBAC |
| US-V06 acceptance (4 paths) | Cloud TTS, music, SFX |

---

## Architecture

```
SCRIPT (approved) → TTS (CPU/espeak) → NARRATION.wav
silent MP4 → FFmpeg mux → narrated MP4 (VIDEO asset)
Export: manifest v3 + narration.wav sidecar per scene
```

**Stop conditions evaluated:** No platform redesign; multi-scene preserved; v1/v2 compat via `NARRATION_ENABLED=false`; Olares espeak viable.
